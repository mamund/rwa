Maze+XML Profile

Describes a common profile for implementing Maze+XML.

  • rel:self = http://alps.io/profiles/maze-xml
  • rel:help = http://amundsen.com/media-types/maze/

maze

Indicates the parent tag of Maze+XML document.
  • id : maze
  • type : semantic
  • cardinality : single
  • appears : SHOULD
  • contained : false
  • Contains:

    • href : #collection
    • type : semantic
    • cardinality : single
    • appears : MAY

    • href : #item
    • type : semantic
    • cardinality : single
    • appears : MAY

    • href : #cell
    • type : semantic
    • cardinality : many
    • appears : MAY

    • href : #error
    • type : semantic
    • cardinality : single
    • appears : MAY

error

Describes an error message.
  • rel:help = http://amundsen.com/media-types/maze/format/#error-element
  • id : error
  • type : semantic
  • contained : true

collection

Describes a collection of mazes.
  • rel:help = http://amundsen.com/media-types/maze/format/#collection-rel
  • id : collection
  • type : semantic
  • contained : true
  • Contains:

    • href : #maze
    • name : maze
    • type : safe
    • rt : cell
    • cardinality : many
    • appears : MAY

item

Describes an individual maze.
  • rel:help = http://amundsen.com/media-types/maze/format/#item-element
  • id : item
  • type : semantic
  • contained : true

cell

Describes a cell in a maze.
  • rel:help = http://amundsen.com/media-types/maze/format/#cell-element
  • id : cell
  • type : semantic
  • contained : true
  • Contains:

    • href : #title
    • type : semantic
    • cardinality : single
    • appears : SHOULD

    • href : #current
    • type : safe
    • rt : cell
    • cardinality : single
    • appears : SHOULD

    • href : #start
    • type : safe
    • rt : cell
    • cardinality : single
    • appears : MAY

    • href : #north
    • type : safe
    • rt : cell
    • cardinality : single
    • appears : MAY

    • href : #south
    • type : safe
    • rt : cell
    • cardinality : single
    • appears : MAY

    • href : #east
    • type : safe
    • rt : cell
    • cardinality : single
    • appears : MAY

    • href : #west
    • type : safe
    • rt : cell
    • cardinality : single
    • appears : MAY

    • href : #exit
    • type : safe
    • cardinality : single
    • appears : MAY

title

Contains the name of the cell. NOTE: This is still missing from the Maze+XML spec! (MCA)
  • id : title
  • type : semantic
  • contained : true

title

Contains the name of a collection, maze, or cell.
  • id : title
  • type : semantic
  • containers : collection maze title

  • id : maze
