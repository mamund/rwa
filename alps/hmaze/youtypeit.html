Yout Type It, We Post It Profile

Describes a common profile for implementing the ytiwpi service.

  • This profile has six transitions (w/ possible semantic elements) and six semantic descriptors (two of which are containers).
  • There is no indication of 'cardinality' or 'appears' here.
  • There is no assurance that any of the descriptors (transition or semantic) will appear in a response. What is listed here are possible descriptors for the 'YouTypeIt' profile.
  • It is possible that some severs will return other semantic descriptors and/or offer other transitions; that's fine. Any clients that are relying on this profile as a guide will likely ignore those unspecified descriptors.
  • rel:self = http://alps.io/profiles/youtypeit

collection

Link that returns a list of messages.
  • rel:help = http://alps.io/iana#collection
  • id : collection
  • type : safe
  • rt : #messageList

item

Link that returns a single message.
  • rel:help = http://alps.io/iana#item
  • id : item
  • type : safe
  • rt : #messageItem

search

Use this transition to search for messages
  • rel:help = http://alps.io/iana#search
  • id : search
  • type : safe
  • rt : #messageList
  • Contains:

    • href : #text
    • type : semantic

    • href : #author
    • type : semantic

    • href : #messageDate
    • type : semantic

post

Use this transition to add a new message to the collection.
  • id : post
  • type : unsafe
  • Contains:

    • href : #text
    • type : semantic

    • href : #author
    • type : semantic

remove

Use this transition to remove a message from the collection,
  • id : remove
  • type : idempotent

update

Use this transition to update an existing message.
  • id : update
  • type : idempotent
  • Contains:

    • href : #text
    • type : semantic

    • href : #author
    • type : semantic

messageList

Represents a list of messages.
  • id : messageList
  • type : semantic
  • Contains:

    • href : #messageItem
    • type : semantic

messageItem

Represents a single message.
  • id : messageItem
  • type : semantic
  • Contains:

    • href : #text
    • type : semantic

    • href : #author
    • type : semantic

    • href : #id
    • type : semantic

    • href : #messageDate
    • type : semantic

text

Contains the text of the message.
  • id : text
  • type : semantic

author

Contains a text string representing the author of the message.
  • rel:help = http://microformats.org/wiki/hcard#fn
  • id : author
  • type : semantic

id

Contains the unique identifier of the message.
  • id : id
  • type : semantic

messageDate

Contains an ISO 8601 date representing the date of the message.
  • rel:help = http://schema.org/Date
  • id : messageDate
  • type : semantic
